# Brazilian Customer Registration

Adds Brazilian tax fields (CPF, RG, CNPJ, Razão Social, Nome Fantasia, Inscrição Estadual, IE
Isento) to the registration + account forms, with CPF/CNPJ checksum validation, input masks,
encrypted storage of sensitive documents, and access-control permissions.

Built on Paymenter's native Custom Properties system — **no core edits**.

Full documentation: [`docs/modules/brazilian-registration.md`](../../../docs/modules/brazilian-registration.md).

**Enable:** Admin → Extensions → *Brazilian Customer Registration* → Install (seeds the fields).
Disabling removes the fields and their stored values.
